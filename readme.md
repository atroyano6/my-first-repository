CHULETA DE COMANDOS DE GIT
pwd: Muestra el directorio actual.
ls -al: Muestra la lista de ficheros en el directorio actual.
clear: limpia la pantalla.
cd <dirname>: Cambia al directorio deseado.
cd ..: Cambia al directorio superior.
mkdir <dirname>: Crea un nuevo directorio.
touche <filename>: Crea un nuevo fichero.
rm <filename>: Borra el fichero.

git clone + url: clona el repositorio del enlace aportado.
git add .: añade a la cesta todos los archivos.
git add <filename>: añade a la cesta el archivo nombrado.
git commit -m "Mensaje a elección con descripción": Prepara la subida al repo remoto.
git push: Sube al repositorio remoto los cambios.

git pull: obtiene los cambios de los compañeros.
